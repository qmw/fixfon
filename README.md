# FixFon — Serwis GSM Warszawa

Przykładowa strona-wizytówka dla serwisu GSM (portfolio stronainternetowa.biz).

## Styl
Bold Pop / Energetyczny: wielka chunky typografia (Exo 2 Black 900), duotone na tle hero,
ukośne przecięcia sekcji, kontrastujące bloki (biel / niebieski / koral / czerń).
Kalkulator ceny wbudowany w hero (desktop).

## Sekcje
Pasek promo → Nav (fixed) → Hero (hasło + kalkulator) → Usługi → Cennik → Jak działamy → Dlaczego my → Opinie → Formularz wyceny + Mapa OSM → Footer

## Stack
Astro 5 + Tailwind CSS 4 (`@tailwindcss/vite`)

## Komendy
```bash
npm install
npm run dev      # serwer deweloperski
npm run build    # produkcyjny build → dist/
npm run preview  # podgląd buildu
```

## Hosting
Projekt hostowany na Vercel (portfolio qmw).
